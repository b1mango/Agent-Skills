# WeChat Article Writer

创建高质量的微信公众号文章，包含研究、写作、标题生成和排版优化。

## 使用场景
- 撰写微信公众号文章
- 生成爆款标题
- 优化文章排版
- 研究热门话题并创建内容

## 工作流程

### 第一步：主题研究
1. 分析目标受众和阅读偏好
2. 研究相关热门话题和关键词
3. 收集数据和案例支撑论点
4. 确定文章角度和核心观点

### 第二步：文章撰写
1. **标题**：生成 5-10 个候选标题
   - 使用心理触发词（好奇、紧迫、利益）
   - 控制在 20 字以内
   - A/B 测试风格标题
2. **开头**：3 秒抓住读者注意力
   - 提出痛点或问题
   - 颠覆认知的数据/观点
   - 讲述引人入胜的故事
3. **正文**：结构清晰，逻辑流畅
   - 小标题分段，每段不超过 3-4 行
   - 使用列表、引用、加粗突出重点
   - 插入数据和案例增加说服力
4. **结尾**：明确的行动号召 (CTA)

### 第三步：排版优化
- 段落间距：15px-20px
- 字号：正文 15-16px，标题 18-20px
- 行间距：1.75-2倍
- 配色方案统一
- 关键段落使用引用框
- 适当使用 emoji 增加视觉层次

### 第四步：最终检查
- [ ] 标题吸引力评估
- [ ] 文章逻辑连贯性
- [ ] 错别字和语法检查
- [ ] 排版在手机端预览效果
- [ ] 原创性检查
- [ ] SEO 关键词密度

```markdown
[具体的 CSS/HTML 排版建议]
```
