# 地缘政治深度分析写作技能

为中文时政深度分析文章提供写作规范和质量控制。目标风格参考《南方周末》深度报道、《三联生活周刊》国际评论、《端传媒》长篇分析。

## 适用场景
- 战争/冲突局势深度分析
- 大国博弈与外交动态解读
- 地缘政治格局演变
- 国际关系与历史因果分析
- 经济制裁与全球经济影响评估

---

## 一、去AI化规则（强制执行）

### 1.1 禁用词与禁用句式

以下词汇和句式在正文中**绝对禁止使用**：

**禁用引导句：**
- 「让我们看一组/看看/来看」
- 「这是什么概念？」
- 「看到了吗？」
- 「这意味着」（用于段首引导时）
- 「没错」「确实如此」
- 「换句话说」
- 「值得一提的是」「值得注意的是」
- 「不得不说」「不可否认」
- 「众所周知」
- 「毫无疑问」「毋庸置疑」

**禁用过渡模板：**
- 「首先…其次…最后…」超过3层的递进结构
- 「第一…第二…第三…」超过3点的编号罗列
- 「一方面…另一方面…」连续成对使用
- 「那么问题来了」
- 「答案是：」「原因很简单：」
- 「我们往下看」「接下来」（用于段尾悬念式引导）

**禁用修辞套路：**
- 连续超过2个反问句
- 反问+自答组合（「为什么？因为……」）
- 「XX？没有。YY？也没有。」否定排比
- 加粗短句单独成段制造节奏感（如「**这就是中东。**」独占一段）

### 1.2 格式限制

| 规则 | 要求 |
|------|------|
| 表格 | 全文不超过3个，且必须是高密度信息表格 |
| 代码块 | 正文中**禁止使用**。不得用代码块展示逻辑链条或流程 |
| 分点列举 | 单次不超过3点；全文分点罗列不超过5处 |
| 段落长度 | 每段不少于80字，鼓励100-200字的长段落 |
| 一句话成段 | **禁止**。每段至少包含2个完整句子 |
| 加粗 | 仅用于人名、组织名、关键术语首引，不用于制造视觉节奏 |
| 引用块 | 仅用于直接引语（某人原话），不用于作者自己的总结 |

### 1.3 句式要求

- **长短交替**：连续3个以上相同长度的句子必须打破节奏
- **主动叙述**：优先使用主语+动词的主动句式，避免被动堆砌
- **因果嵌套**：用自然语言表达因果，如「正是因为A导致了B，而B的直接后果是C」，而不是分三行分别陈述A、B、C
- **比喻和细节**：用具体的比喻和场景细节增加画面感，避免抽象空话

## 二、分析写作规范

### 2.1 推理链条要求

每一个分析性论点必须包含完整的推理链条：

**正确示范**：
> 伊朗之所以选择在3月8日宣布"暂时关闭"霍尔木兹海峡，而非此前外界普遍预期的立即全面封锁，根本原因在于德黑兰的决策层清楚认识到，全面封锁会同时切断伊朗自身剩余的石油出口通道——尽管在制裁下这一通道已萎缩至每日不足50万桶，但仍是维系战时财政的生命线。"暂时关闭"这一措辞精心设计，既向华盛顿释放了"我有能力让你痛"的威慑信号，又为后续谈判保留了升级或降级的弹性空间。

**错误示范（禁止）**：
> **伊朗宣布"暂时关闭"霍尔木兹海峡。**
>
> 为什么是"暂时"？
>
> 因为：
> 1. 全面封锁会切断自身出口
> 2. 保留谈判筹码
> 3. 释放威慑信号
>
> **这就是伊朗的精明之处。**

### 2.2 事实与分析的区分

- **事实陈述**：必须附带来源和日期，如「根据路透社3月10日报道」「国际原子能机构2026年第一季度报告显示」
- **分析推断**：必须使用限定表述，如「由此可以推断」「这意味着在最乐观的估计下」「从历史经验来看」
- **不确定信息**：明确标注，如「目前尚无法独立证实」「多方消息来源存在矛盾」

### 2.3 历史引用原则

- 历史材料**服务于当下分析**，不做百科式教学
- 历史段落不超过全文的15%
- 每引用一段历史，必须在同一段落或下一段落中**连接回当下议题**
- 避免「让我们回到XX年」式的时间跳跃引导

### 2.4 数据引用规范

- 数据必须标注**来源机构+时间**
- 估算数据标注为「据XX估计」或「约」
- 避免堆砌超过5个数字/百分比在同一段中
- 数据服务于论点，不为展示数据而展示

## 三、文章结构规范

### 3.1 整体结构

建议采用以下结构（可灵活调整）：
1. **开篇**：用一个具体场景或事件切入，不超过800字
2. **背景/根源**：历史因果链条，精炼到只保留直接相关的部分
3. **核心分析**：按主题（而非时间线）组织的深度分析段落
4. **外部影响**：地缘/经济/社会冲击分析
5. **前瞻**：基于分析的推演，区分不同情景的可能性
6. **收束**：简短有力，避免鸡汤式结尾

### 3.2 段落间过渡

- 用叙述性语句自然过渡，不用「接下来看…」「让我们转向…」
- 段落首句承接上段末尾逻辑，形成论述的连贯河流
- 允许使用小标题分隔大的主题板块，但同一板块内段落之间不应有标题

### 3.3 标题层级

- 一级标题（#）：仅用于文章总标题
- 二级标题（##）：主要章节划分，不超过6个
- 三级标题（###）：章节内细分，谨慎使用
- 四级及以下标题：**禁止使用**

## 四、质量检查清单

完成写作后，逐项检查：

- [ ] 全文无禁用词/禁用句式
- [ ] 表格不超过3个
- [ ] 无代码块
- [ ] 无一句话成段
- [ ] 分点罗列总数不超过5处
- [ ] 每个分析论点有推理链条
- [ ] 数据均标注来源和时间
- [ ] 历史段落不超过全文15%
- [ ] 段落间过渡自然流畅
- [ ] 长短句交替，无节奏单一的段落
- [ ] 整体字数在目标范围内
